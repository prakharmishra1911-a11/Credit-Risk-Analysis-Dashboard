# Credit Risk Analysis Dashboard (Power BI)

## 📊 Project Overview
This project focuses on analyzing customer data to identify credit risk patterns and potential default indicators.

## 🔍 Key Insights
- ~29% customers are high-risk
- High-risk customers mostly have low employment duration
- Majority of risky customers lie below average credit amount
- Owned housing customers show lower risk

## 🛠 Tools Used
- Power BI
- Data Visualization
- Basic DAX
